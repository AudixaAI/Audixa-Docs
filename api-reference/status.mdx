---
title: "GET /status"
api: "GET https://api.audixa.ai/v2/status"
description: "Check the status of a text-to-speech generation job"
---

Retrieves the current status of a previously submitted TTS job. Once the status is `Completed`, the response includes an `url` to download the generated audio.

## Request

<ParamField query="generation_id" type="string" required>
  The generation ID returned from [POST /tts](/api-reference/tts).
</ParamField>

## Response

<ResponseField name="status" type="string">
  Current status of the generation job:
  - `Generating` - Audio is still being processed
  - `Completed` - Audio is ready for download
  - `Failed` - Generation failed (see `detail` for reason)
</ResponseField>

<ResponseField name="url" type="string">
  URL to download the generated MP3 file. Only present when `status` is `Completed`.
</ResponseField>

<ResponseField name="detail" type="string">
  Error message. Only present when `status` is `Failed`.
</ResponseField>

## Examples

<RequestExample>
```bash cURL
curl -X GET "https://api.audixa.ai/v2/status?generation_id=gen_abc123xyz789" \
  -H "x-api-key: YOUR_API_KEY"
```

```python Python
import audixa

audixa.set_api_key("YOUR_API_KEY")

generation_id = "gen_abc123xyz789"

# Check status of a generation
status = audixa.status(generation_id)
print(f"Status: {status['status']}")

if status["status"] == "Completed":
    print(f"Audio URL: {status['url']}")
elif status["status"] == "Failed":
    print(f"Error: {status['detail']}")
```

```javascript JavaScript
import Audixa from 'audixa';

const audixa = new Audixa('YOUR_API_KEY');

const generation_id = 'gen_abc123xyz789';

// Check status of a generation
const status = await audixa.getStatus(generation_id);

console.log('Status:', status.status);

if (status.status === 'Completed') {
  console.log('Audio URL:', status.audio_url);
} else if (status.status === 'Failed') {
  console.log('Error:', status.detail);
}
```

```typescript TypeScript
import Audixa, { TTSStatusResponse } from 'audixa';

const audixa = new Audixa(process.env.AUDIXA_API_KEY!);

const generation_id = 'gen_abc123xyz789';

const status: TTSStatusResponse = await audixa.getStatus(generation_id);

if (status.status === 'Completed' && status.audio_url) {
  console.log('Audio URL:', status.audio_url);
} else if (status.status === 'Failed') {
  console.log('Error:', status.detail);
}
```
</RequestExample>

## Polling Best Practices

<CardGroup cols={2}>
  <Card title="Poll Interval" icon="clock">
    Poll every 1-2 seconds. Don't poll faster than once per second.
  </Card>
  <Card title="Timeout" icon="hourglass">
    Set a maximum timeout (e.g., 60 seconds) to avoid infinite loops.
  </Card>
  <Card title="Error Handling" icon="triangle-exclamation">
    Always handle the `Failed` status gracefully with retry logic.
  </Card>
  <Card title="Cache Results" icon="database">
    Once completed, cache the `url` to avoid re-polling.
  </Card>
</CardGroup>

## Error Responses

<Accordion title="401 Unauthorized">
Missing or invalid API key.

```json
{
  "detail": "Invalid or missing API key"
}
```
</Accordion>

<Accordion title="404 Not Found">
Invalid or expired generation ID.

```json
{
  "detail": "Generation not found"
}
```

<Note>
  Generation IDs expire 24 hours after creation.
</Note>
</Accordion>

<Accordion title="429 Too Many Requests">
Rate limit exceeded.

```json
{
  "detail": "Rate limit exceeded. Please retry after 30 seconds."
}
```
</Accordion>
