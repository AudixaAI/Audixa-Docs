## Get started in three steps

Get up and running with Audixa’s TTS API in minutes. Introduce the 2-step async workflow.

### Step 1: Sign Up & API Key

To begin, [create a free account](https://audixa.ai/signup?utm_source=Documentation) on the [Audixa website](https://audixa.ai).

Every new account includes 10,000 free credits to try out the service.


### Step 2: Generate Your First Speech

You can generate high-quality speech directly from the web dashboard without writing any code.

1. Visit the TTS Dashboard.
2. Select Base Model
3. Select a preset text like 'Casual Conversation'
4. Click the generate button
   
Your audio will be ready in a few seconds, and you can listen or download it directly from the page.
